۰ = نامحدود
فقط «VLESS (WebSocket)» با سرور داخلی همین پنل روی Railway کار می‌کنه. بقیه‌ی گزینه‌ها فقط وقتی کار می‌کنن که یه سرور Xray/V2Ray جدا (VPS) داشته باشی.
با سرور داخلی Railway، پورت همیشه ۴۴۳ هست و نیازی به پر کردن نداری.
این مقدار ثابته و قابل تغییر نیست.

تنظیمات سرور

مقادیر پیش‌فرض برای ساخت کانفیگ‌ها

این پنل حالا خودش سرور Xray رو (روی همین Railway) اجرا می‌کنه. توی «آدرس/دامنه سرور» همون دامنه‌ی Railway (مثل b86.up.railway.app) رو بذار. پورت و امنیت خودکار روی ۴۴۳/TLS ثابته چون از دامنه‌ی Railway استفاده می‌شه؛ «مسیر (Path)» رو هرچی بذاری، همون برای اتصال کاربرا استفاده می‌شه.
این مقدار ثابته و قابل تغییر نیست.

Reality (سرعت بالا / ضدفیلتر)

اتصال مستقیم بدون عبور از edge ریلوی - نیازمند یک TCP Proxy جدا

۱) توی تنظیمات Railway، بخش Networking، یک «TCP Proxy» جدید بساز که به پورت 48281 (یا هرچی این‌جا ست کردی) اشاره کنه. ۲) هاست و پورتی که Railway بهت داد (چیزی مثل xxxx.proxy.rlwy.net:12345) رو پایین وارد کن. ۳) دکمه‌ی «ساخت کلید» رو بزن. ۴) فعالش کن و ذخیره کن.
